{{ site.title }}

{{ site.title }} ({{ site.data.github.repo }}) is a customizable navigation menu for Spree. Sure you can overwrite the view file in your application to add your own menu items. But what fun would that be?

Works well with multi-tenant sites or applications where overwriting the views/spree/shared/_main_nav_bar.html.erb file is not an option.

Installation

Add this line to your Spree application's Gemfile:

gem '{{ site.data.github.repo }}', github: '{{ site.data.github.username }}/{{ site.data.github.repo }}'

Run the bundle command to install it:

bundle install

After installing, run the generator:

bundle exec rails g {{ site.data.github.repo }}:install

Testing

Generate a dummy application

bundle exec rake test_app

Running tests

bundle exec rake spec
